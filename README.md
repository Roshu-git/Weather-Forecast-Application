weather forecast application based on the weather condition.

## Setup Instructions

Follow these steps to run the Weather Forecast Application on your local machine.

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/weather-forecast-app.git
```

### 2. Open the Project

Navigate to the project folder:

```bash
cd weather-forecast-app
```

### 3. Open in Visual Studio Code

Open the project using Visual Studio Code.

### 4. Install Live Server (if not already installed)

- Open the **Extensions** panel in VS Code.
- Search for **Live Server**.
- Install the extension by Ritwick Dey.

### 5. Get a WeatherAPI Key

1. Visit **https://www.weatherapi.com/**
2. Create a free account.
3. Copy your API key.
4. Open `assets/js/custom.js`.
5. Replace the existing API key with your own:

```javascript
const apiKey = "YOUR_API_KEY";
```

### 6. Run the Project

- Right-click on `index.html`.
- Select **Open with Live Server**.

Or open:

```
http://127.0.0.1:5500
```

### 7. Allow Location Access

When prompted by the browser, click **Allow** to enable current location weather.
